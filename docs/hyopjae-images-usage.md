# 협재 해수욕장 무장애 이미지 활용 구상

협재 해수욕장 무장애 데이터(`public/제주특별자치도_무장애여행정보_협재해수욕장/`)에 포함된 CSV와 이미지의 앱 적용 현황과 추가 활용 방안을 정리합니다.

---

## 1. 현재 적용 내용

### 1.1 CSV 적용

- **MAP_REGIONS**에 **협재 해수욕장** 지역 추가
  - `value`: `hyopjae-협재해수욕장`
  - `file`: `010-2.csv` (장소명·상세·이미지파일명 포함)
  - `format`: `hyopjae` (기존 region CSV와 컬럼 구조가 다름)
- **홈 탭**: 협재 해수욕장 리뷰 카드에 해당 지역 CSV 기반 **무장애 뱃지**(경사로, 휠체어, 화장실 등) 표시
- **지도 탭**: 지역 선택에서 「협재 해수욕장」 선택 시 `010-2.csv` 기준으로 마커·정보카드 표시

### 1.2 이미지 활용 (현재 구현)

- **지도 정보카드**: 협재 해수욕장을 선택한 뒤, 마커에 마우스를 올리면 뜨는 **정보카드**에 해당 장소의 이미지(`images/010_DSC00xxx.jpg`)를 표시
  - CSV의 `이미지파일명` 컬럼과 `imageBaseUrl`을 조합해 카드 내 `<img>`로 노출

---

## 2. 추가 이미지 활용 구상

### 2.1 홈·리뷰 카드에서 활용

- **협재 해수욕장 리뷰 카드**에 “이 지역 무장애 포인트” 썸네일 1~2장 노출  
  - 해당 `regionKey`의 CSV에서 이미지가 있는 행만 필터해, 대표 1~2개 이미지를 카드 하단 또는 뱃지 옆에 작게 표시
- **장소별 상세 모달** (추가 개발 시)  
  - 협재 해수욕장 내 특정 장소(예: 1번 주차장, 휠체어 화장실)를 선택했을 때, 해당 행의 이미지를 상세 설명과 함께 큰 이미지로 표시

### 2.2 지도·탐색 경로에서 활용

- **마커 아이콘**  
  - 이미지가 있는 장소는 기본 빨간 마커 대신, 해당 장소 이미지를 작게 잘라 **커스텀 마커**로 사용 (접근성·휠체어 구간 등 시각적 구분)
- **경로/코스 뷰** (010-1.csv 연동 시)  
  - `010-1.csv`는 위·경도·고도만 있으므로 **경로 폴리라인**으로 그린 뒤, 경로 상 주요 지점에 010-2.csv와 매칭되는 이미지를 **경로 갤러리** 형태로 슬라이드 또는 목록으로 표시

### 2.3 갤러리·온보딩

- **협재 해수욕장 전용 갤러리**  
  - `images/` 폴더의 11장을 한 페이지에서 그리드 또는 스와이프 갤러리로 보여주고, 각 이미지에 CSV 행과 연결해 장소명·상세 설명 툴팁 또는 하단 캡션 표시
- **무장애 여행 소개/온보딩**  
  - 앱 첫 실행 또는 “무장애 여행이란” 화면에서 협재 해수욕장 사진 몇 장을 예시로 사용해, 실제 무장애 시설 사진과 함께 안내

### 2.4 접근성·공유

- **이미지 대체 텍스트**  
  - 모든 무장애 이미지에 `alt`를 넣어, CSV의 장소명·상세정보를 반영 (예: “협재 해수욕장 1번 주차장 전경”)
- **공유/저장**  
  - 장소 상세 또는 갤러리에서 “이미지 저장” / “링크 공유” 시 해당 무장애 포인트 URL과 함께 이미지 첨부
