# Jeju Reviews — Vite + React + Tailwind

간단한 데모 앱입니다 (제주도 특화 리뷰 & 지도 뷰). 프로젝트를 로컬에서 실행하려면:

1. 의존성 설치

```bash
npm install
```

2. 개발 서버 실행

```bash
npm run dev
```

브라우저에서 표시되는 로컬 URL (예: http://localhost:5173)를 열면 앱을 확인할 수 있습니다.

참고:
- Tailwind가 포함되어 있으며 기본 구성만 제공합니다.
- 이미지 외부 URL을 사용하므로 네트워크가 필요합니다.
